# InTrack
InTrack is a modern, scalable inventory management system designed for efficient stock tracking, order management, and analytics. Built with Next.js, Tailwind CSS, Prisma, Supabase, Firebase Auth, Cloudinary, and Redis for optimized performance and flexibility.

## GitHub Repository

Check out the source code for this project on [GitHub](https://github.com/lewisbagay/intrack).

## Project Status
This project is currently in the early stages of development. Initial dependencies and environment setups have been configured, and work is progressing towards implementing core features like inventory tracking and order management.
